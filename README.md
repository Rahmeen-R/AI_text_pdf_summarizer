AI Text & PDF Summarizer

A Streamlit‑powered AI tool for summarizing text and PDF documents using Gemini 2.5 Flash.

The tool allows users to upload text or PDF files and generate clear, concise summaries in multiple lengths and languages. Summarize text or PDF documents instantly

Choose summary length: Short, Medium, Long

Clean, responsive Streamlit interface

Fast, high‑quality summaries powered by Gemini 2.5 Flash

How to Run:

The summarizer uses Google’s Gemini API, and the API key is loaded securely from a .env file. Simply create this file in the project folder and add GEMINI_API_KEY=your_key_here. The app reads this key automatically at startup. After installing the required packages from requirements.txt (Streamlit, PyPDF2, python‑dotenv, and the Google GenAI client), you can launch the tool by running streamlit run app.py. This starts the interface in your browser, where you can upload text or PDF files and generate summaries instantly.
