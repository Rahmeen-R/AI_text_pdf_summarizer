AI Text & PDF Summarizer

A Streamlit‑powered AI tool for summarizing text and PDF documents using Gemini 2.5 Flash.
This project demonstrates my ability to build practical AI applications with clean UI, efficient prompt engineering, and seamless LLM integration. The tool allows users to upload text or PDF files and generate clear, concise summaries in multiple lengths and languages. It reflects strong skills in AI development, Streamlit UI design, and PDF processing.


Features:

Summarize text or PDF documents instantly

Choose summary length: Short, Medium, Long

Supports English and Urdu

Clean, responsive Streamlit interface

Fast, high‑quality summaries powered by Gemini 2.5 Flash

Run Method:

Install dependencies using pip install -r requirements.txt, add your GEMINI_API_KEY in a .env file, then run the app with streamlit run app.py to launch the summarizer in your browser.
